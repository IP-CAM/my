出色的ETshop前台功能

ETshop的前台之所以有如此好的体验及众多的操作，是离不开这些优秀的操作功能，才带来如此出色的ETshop前台
  • 可视化装修

    商城首页、店铺首页以及专题页均可自定义可视化装修多套模板，可选多种样式自由化装修自定义

  • 多城市切换

    ETshop可以进行不同的城市切换，并且根据相应的城市显示不同的商品及库存

  • 顶级分类页

    后台设置banner广告位，新品首发，热销商品，根据分类显示楼层

  • 地区列表5级

    ETshop拥有国家标准三级地区，运营商还可以手动输入四级五级区域地址

  • 仿京东白条

    ETshop开发仿京东白条功能，消费者可以使用白条分期购买大额商品

  • 推荐分成

    推荐商品并促成交易即可按照后台设定的分成比例获取佣金

  • 物流跟踪

    每笔订单实时物流跟踪，信息准确，提前预知收货时间

  • 多区域多仓库

    支持多区域、多仓库运营模式设置，实现送货地址与库存点快速关联

  • 模糊搜索

    可智能搜索关键字的同义词，提高精确性，让客户轻松找到他想要的商品

丰富的商品功能

ETshop平台丰富的商品功能，为消费者的购物，以及在商城的整体操作，都带来了优质的体验
  • 商品二维码

    扫描商品详情二维码可直接
    购买，并可分享给手机好友

  • 商品详情对比

    可将多个商品进行详情、价
    格等多重条件对比

  • 商品分期购买

    分期购买可以缓解购买高额
    商品的压力，刺激购买欲望

  • 商品提货卡

    线下促销发放商品提货卡
    用户可线上进行等额提货

  • 商品扫码入库

    扫码枪扫描商品一键加入后
    台库存，管理更便捷

  • 商品退换货

    商城支持随时退、先行赔等
    更多保障消费者利益

  • 商品组合购买

    后台可设置与商品关联的相
    关配件，前端自动推荐组合

  • 商品库存管理

    后台全程跟踪商品出入库信
    息，商品库存一目了然

可视化的编辑操作，带来更便捷的体验

为了商家更好的编辑自己喜爱的店铺风格，ETshop专门推出了店铺可视化编辑功能，入驻的商家在修改店铺的时候，可以实时预览店铺修改的效果，并且一键保存，并可在前台生成自己预览时的效果

多套精美店铺模板，让更多商家秀出个性

当入驻的商家还在为不懂审美，不懂装修，也没有美工的情况下，ETshop还贴心推出了多套精美的商城模板选择
每一套模板都是经过ETshop的设计师精心挑选，不仅符合大众审美，也紧跟现代电商潮流

海量促销功能，引爆全网销量

促销是搭建电商平台不可或缺的重大模块，ETshop作为一款全新的B2B2C2C系统，非常重视此项功能
所以在所有的促销模块中精心打造出了多种强大的促销功能来助您的平台销量

集成多种主流支付方式

便捷好用的支付，是一个商场的核心要素，拥有好的支付体系，让用户减少付款的繁琐，方能快速促成下单

多样化的平台配送方式

  • 发展平台自营快递

    运营商可以将自己的平台，像京东或者1号店一样，发展自营快递服务，做出平台自己的特色，不仅利于线下品牌的传播，也让更多用户喜爱上在平台享受自营产品的购物

  • 第三方快递公司合作

    平台不仅可以发展自营快递，也可以与第三方快递公司谈合作，全面开展与顺丰、圆通、韵达、百世等多方公司的合作，也来让商城购物的用户对配送方式能够做出更多选择

ETshop对接的生态平台及产品

ETshop集合市场众多电商经验人士提出建议反馈，再结合电商市场的最新动态，开发出了众多商家最常用的电商热门功能，丰富多样，带来更多精彩
